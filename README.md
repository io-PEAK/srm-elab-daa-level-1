## **What This Repo Contains**

- **75 Level–1 DAA solutions** from [SRM Elab](https://dld.srmist.edu.in/)
- **C (`.c`) and C++ (`.cpp`) solutions** for each problem
- File names begin with the start of each question's first line, for fast searching.

## **How to Find an Answer**

1. **Copy the first few words of your SRM Elab question.**
2. **Search in this repo** (use the GitHub search bar, press `t` for file finder).
3. Match the file name inside the appropriate topic folder.
4. Open the file for the code solution.

> [!IMPORTANT]
>
> - If your answer isn’t accepted:
>   - Check the file extension (`.c` or `.cpp`)
>   - In Elab, select the correct language:  
>     &nbsp;&nbsp;• `.c` → **C**  
>     &nbsp;&nbsp;• `.cpp` → **C++**
>   - Change the language before running/submitting.

## **Enable Copy in SRM Elab**

If you are unable to copy code directly inside SRM Elab due to right-click restrictions, use this browser extension:

> [Absolute Enable Right Click & Copy - Chrome Web Store](https://chromewebstore.google.com/detail/absolute-enable-right-cli/okaeekdeohehbdanbbohcfhhlblcmjkm)
>
> - Login in the SRM Elab.
> - Press the **"Enable Copy"** option in the extension.
> - Refresh the Elab page.
> - **Do not press the "Infinity Mode"** option.
>
> This allows right-click and copy on websites that block these actions.
